# S/4HANA ABAP RAP Portfolio

This repository contains hands-on SAP S/4HANA ABAP development projects built using modern SAP technologies including RAP (RESTful ABAP Programming Model), CDS View Entities, OData V4, and Fiori Elements.

The objective of this portfolio is to demonstrate practical implementation of business applications using SAP's modern development approach and clean architecture principles.

---

## 🚀 Featured Project

### Purchase Order Management Application

An end-to-end Purchase Order Management solution developed using SAP RAP and Fiori Elements.

#### Key Features

* Purchase Order Header and Item Management
* Automatic Purchase Order Number Generation using SNRO
* OData V4 Service Exposure
* Fiori Elements List Report and Object Page
* Vendor Validation
* Material Validation
* Unit Validation
* Currency Validation
* Purchase Order Date Validation
* Automatic Total Amount Calculation
* Submit Purchase Order Action
* Status-Based Business Process Control
* Dynamic UI Behavior using Instance Feature Control
* Automatic Creation Timestamp Management
* Header-Item Composition Model

#### Technologies Used

* ABAP
* CDS View Entities
* RAP Managed Scenario
* OData V4
* Fiori Elements
* Behavior Definitions
* Determinations
* Validations
* Actions
* Instance Feature Control
* Number Range Objects (SNRO)
